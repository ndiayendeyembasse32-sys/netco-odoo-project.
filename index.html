Réalisations techniques

Ruban flottant latéral dynamique

Menu d'action rapide fixé sur le bord droit de l'écran, avec formes en clip-path polygone. Un listener JavaScript détecte le scroll : au-delà de 40px, le ruban passe en mode compact, et se redéploie au survol.

HTML CSS clip-path JavaScript scroll position: fixed

ruban-flottant.js
// Scroll listener — mode compact (function() { var ruban = document .getElementById('ruban-flottant-netco'); if (ruban) { window.addEventListener( 'scroll', function() { if (window.pageYOffset > 40) { ruban.classList.add('compact'); } else { ruban.classList.remove('compact'); } }); } })(); /* CSS — clip-path polygone */ .ruban-btn { clip-path: polygon( 15px 0%, 100% 0%, 100% 100%, 15px 100%, 0% 50% ); width: 230px; transition: width 0.3s ease; } .compact .ruban-btn { width: 45px; }

Formulaire de devis personnalisé

Reskin complet du formulaire Odoo Website Form : fond vert arrondi, champs en forme de pilule, cases à cocher dans un encadré translucide, bouton blanc avec texte vert. Zéro modification du backend, 100% CSS injecté.
